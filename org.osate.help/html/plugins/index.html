OSATE plug-ins

OSATE also provide a set of plug-ins, either for validating your models or interface them with other tools. Users can develop their own plug-ins to connect AADL models with other tools. The following sub-sections present the tools included in the pre-defined set of plugins shipped with OSATE. Users interested in developing their own plug-ins would read the plug-in help development.

The following plug-ins are currently included with OSATE:

  • Instance Model Viewer or IMV: provides a graphical view of the Instance Model.
  • SEI analysis plug-ins: analyzes models for given quality criteria (security, safety, performance, etc.).
  • LUTE: provides a high-level language for querying the model and validating requirements without having to write a complete OSATE plug-in.
  • Simulink and SCADE importers: provides the ability to generate an AADL skeleton model from a functional model that is using SCADE or Simulink language.
